Future<List<NQSurahTitle>> getSurahList()

Returns a list of SurahTitles containing the surah title and relevant information SurahTitle contains: - number: index of the surah - name: arabic name - transliterationEn: english transliteration of the arabic name - translationEn: english translation of the name - totalVerses: total number of verses present in the surah - revelationType: revelation type - whether makkan or madani surah Parameter: None Returns: List of SurahTitle

Implementation

static Future<List<NQSurahTitle>> getSurahList() async {
  String surahListJsonStr = await rootBundle.loadString(NQPathConstants.surahListPath);
  List<NQSurahTitle> surahFromJson =
      List<NQSurahTitle>.from(json.decode(surahListJsonStr).map((x) => NQSurahTitle.fromJson(x)));
  return surahFromJson;
}
